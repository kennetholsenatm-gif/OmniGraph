# OmniGraph wiki

This folder serves two audiences:

1. **Browsing the repo** — Use the links below; they point at the `main` branch in this same repository.
2. **GitHub “Wiki” tab** — Wikis use a separate git repo (`OmniGraph.wiki.git`). To populate that tab from these files, follow **[SYNC](SYNC.md)** (clone the wiki, copy `wiki/*.md`, push).
