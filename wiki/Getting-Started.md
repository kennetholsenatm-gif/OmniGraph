## Quick start (web)

From **`packages/web`**, install dependencies and start the dev server—see **[Contributor commands](https://github.com/kennetholsenatm-gif/OmniGraph/blob/main/docs/development/contributor-commands.md)**.

Open the dev URL Vite prints, then use the sidebar to explore the workspace.
