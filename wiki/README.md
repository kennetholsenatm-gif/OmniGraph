# Wiki source (companion to GitHub Wiki)

Markdown in this directory is meant to be copied into the repository’s **GitHub Wiki**, or kept as the canonical wiki text in the main repo.

- Edit pages here and merge via normal pull requests.
- To publish on GitHub: enable Wiki on the repo, then create pages whose titles match the filenames (e.g. `Home`, `Control-plane-CLI`).

`Home.md` is the usual entry point. `_Sidebar.md` is the standard GitHub Wiki sidebar page when pasted as a page named `_Sidebar`.
